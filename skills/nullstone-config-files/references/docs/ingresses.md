# `ingresses` top-level element

An ingress is a block that represents a device that allows access from the public internet to a private resource. Some examples are load balancers, API gateways, and reverse proxies.

An ingress has the same schema as a [`block`](https://docs.nullstone.io/gitops/iac/blocks.html).
